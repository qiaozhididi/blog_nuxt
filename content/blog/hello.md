# Hello Obsidian

# 项目综合介绍

## 项目概述
本项目是一个基于 Obsidian 的知识管理与博客发布模板，旨在帮助用户将 Obsidian 中的笔记快速转换为可发布的博客文章。通过标准化的前置元数据（Front Matter）与简洁的 Markdown 结构，实现“写一次、随处发布”的工作流。

## 核心功能
- **一键元数据**：自动在文章顶部插入 `title`、`description`、`date` 等关键信息，保证 SEO 与阅读体验。
- **格式兼容**：严格遵循 GitHub Flavored Markdown 规范，支持代码高亮、表格、任务列表等扩展语法。
- **主题友好**：不依赖特定主题，输出纯净 Markdown，可无缝接入 Hugo、Hexo、VitePress 等静态站点生成器。
- **Obsidian 原生支持**：充分利用 Obsidian 的插件生态（如 Git、Publish、Advanced Tables），让写作与版本管理同步进行。
- **多平台发布**：通过 CI/CD 脚本，可将同一篇文章自动同步到博客、知乎专栏、公众号草稿箱等多个平台。
